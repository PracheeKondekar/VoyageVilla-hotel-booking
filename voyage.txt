*********require listing.js in app.js{
    1. In listing.js
You wrote:
const Listing = mongoose.model("Listing", ListingSchema);
module.exports = Listing;
Here:
Listing is the model name you are exporting.
Whatever you export, that is what app.js receives.
So you exported a model named Listing.
✅ 2. In app.js
You imported it:
const Listing = require("./models/listing");
Here:
You are storing the imported model inside a variable also called Listing.
This is why you use:
new Listing({...})
}
